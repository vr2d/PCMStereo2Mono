# PCMStereo2Mono
Convert PCM data from stereo to mono.

##Usage
  Usage: pcms2m [options] <file>
  
  Options:
  
    -b --bits-per-sample  PCM's bits per sample, 8 or 16, default is 16

    -o --output-file      file path to save

    -f --file             PCM file
